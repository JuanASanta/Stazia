#**STAZIA**

Una aplicación que le devuelve a los anfitriones la gestión de la estancia de sus Inquilinos o Clientes dándoles la oportunidad de desligarse de Booking o Airbnb. Una aplicación de Gestión de reservas para pequeños o medianos negocios que no tienen presupuesto o necesidad de gestionarse como si fuera un Hotel, pero quieren tener un control y una autonomía de los típicos sitios que te cobran un pastizal por gestionarte esas cosas.
